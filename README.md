## Deploy 
- Click Here :- [main repo](https://github.com/LEGEND-AI/LEGENDBOT)
- Fork The Main Repo 
- Go to Setting Of Repo 
- Rename the Fork Repo Put Anything Example : LEGEND1BOT, HSKSBOT, LEGENDOFFBOT etc...
- Now U Can Deploy This Bot
